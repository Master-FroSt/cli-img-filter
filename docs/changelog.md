# Changelog
File ini berisi perjalanan aplikasi selama pengembangan, termasuk komentar penulis dan hal yang akan dilakukan selanjutnya.

## Keresahan
Ide pembuatan image filter berasal dari kursus programming di [Brilliant.org](https://brilliant.org/home/). Saat pertama kali mengetahui manipulasi gambar dilakukan dengan perulangan sederhana, aku langsung tertarik membuatnya. Aku pernah bikin versi GUI juga, tapi kodenya rumit jadi kubuat CLI saja.

Ketika mencari tahu soal filter gambar, aku menemukan library OpenCV untuk manipulasi gambar lebih jauh. Mungkin akan kuimplementasikan nanti. Begitu selesai, aku akan memiliki fungsi praktis dari aplikasi ini, yaitu membuat outline dari gambar karakter game atau anime dengan edge detection. Hasilnya akan kubuat sebagai bahan _coloring_.
